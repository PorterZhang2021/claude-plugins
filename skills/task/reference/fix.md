# fix — Bug 修复任务清单格式

**第一个任务必须是能复现 Bug 的失败测试：**

```markdown
## Task 1：复现测试（必须先完成）

- [ ] **[测试]** `<测试文件路径>`（先查看项目已有测试文件结构，遵循已有约定；若无，询问用户偏好）
  - `test_<bug_description>_reproduces_the_bug` ← 必须先失败
- [ ] **[修复]** `<file_path>`（复现测试红色后开始）
  - 具体修改点

## Task 2：回归测试补充（可选）

- [ ] 补充边界条件测试，防止类似 Bug 复发
```
